## About <a name="about"></a>


**Gourmet2Go** is a full-stack web app that was designed specifically for the Sault College community. It provides an easier way for
students to order from **Gourmet2Go** which is run by the Sault College culinary program. This app provides a streamlined, user-friendly 
platform for ordering student-made meals. Furthermore, it gives students an easy way to browse menus, check stock availability, and place 
orders. Additionally, we have QR codes for orders and an Admin QR-code scanner to make picking up orders easier and quicker. The app also 
has a secure role-based access system which makes some pages unreachable if you are not authenticated or not authorized. Overall, the 
**Gourmet2Go** web app handles everything from the kitchen to the customer.


## Motivation <a name="motivation"></a>


Currently, Gourmet2Go requires customers to wait in long lines to order in-person. While waiting in line, the customer has no clue if the 
dish they want is even in stock. With the introduction of our app, customers can go online and see the live stock for every item in the
menu. Then, they can place their order online. The pickup process will also be much faster and easier due to all the orders being shown in 
the app with order numbers and QR codes for quick look up.


## Key Features <a name="key-features"></a>


### For Users


- Menus
- Real-time stock
- QR code pick up system
- Cart management (item limit, persistence, instant total calculation)
- Personalized greeter
- Order history
- Leave reviews 


### For Admins


- Admin Control Panel
- Menu CRUD 
- Dishes CRUD 
- Gallery CRUD
- QR Scanner
- Live order tracking
- User moderation
- Analytics
- Reviews
- Inbox


### Technical Highlights


- Progressive Web App (PWA)
- WASM image processing
- Role-Based Access Control (RBAC)
- Dark mode
- Supabase Realtime
- Flask on Vercel Serverless for banning and deleting users


## Getting Started <a name="getting-started"></a>


This application uses React with Vite for the frontend, Supabase as the Backend as a Service (BaaS), and a small Flask serverless API. 
You will need Node.js and Python installed on your machine before you can run the app locally. You will also need to setup Supabase. You
can learn more by going to our `SETUP.md` 
<a href="https://github.com/iraana/Capstone-Project/blob/main/docs/SETUP.md">click here</a>.
